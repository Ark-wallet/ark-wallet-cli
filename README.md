# Ark Wallet CLI ⚡

⚠️**The Wallet is still WIP**

Ark wallet with CLI

### Features

- Self-custody
- Built on Bitcoin (non-soft fork)
- Open source
  
### How works?

- Create your own keys
- Then generate address
- Accept payment
